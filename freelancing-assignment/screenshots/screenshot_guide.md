# Screenshot Collection Guide

This checklist ensures you capture all required visual evidence for the assignment's PDF submission.

## 1. Khamsat Screenshots
- [ ] Account Dashboard showing your username and profile picture.
- [ ] Published Service Page (The "Custom AI Chatbot Integration" service - must be visible and active).
- [ ] Profile Settings page showing completeness.

## 2. Mostaql Screenshots
- [ ] Public Profile View showing your professional photo, Arabic headline, and bio.
- [ ] Skills Section expanded.
- [ ] Portfolio Grid showing the 3 uploaded projects.

## 3. Freelancer.com Screenshots
- [ ] Public Profile View (Headline, Photo, Bio).
- [ ] Verification badges (Identity, Payment, etc.).
- [ ] Portfolio Section.

## 4. Upwork Screenshots
- [ ] Full Top Section (Profile Headline, Photo, Job Success Score/Completeness).
- [ ] Expanded Profile Bio.
- [ ] Skills & Certifications Section.
- [ ] Portfolio Section showing AutoAnalyst AI, ADPilot, etc.

## 5. PeoplePerHour Screenshots
- [ ] Profile Cover Image & Headshot.
- [ ] "About" Section.
- [ ] Active "Offers" Section.

## 6. Portfolio Projects Screenshots
*(Capture these locally from your projects to embed in the PDF)*
- [ ] AutoAnalyst AI: Dashboard UI & AI Chat.
- [ ] ADPilot: Campaign Creation Screen & Analytics.
- [ ] AI Automation Platform: Node-based workflow builder.
