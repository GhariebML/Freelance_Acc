Body Performance Intelligence System

End-to-end Machine Learning system for physical performance analytics.

The Challenge

Athletes generated massive amounts of raw IoT and manual fitness data, but coaches lacked actionable insights to prevent injuries and optimize training loads.

The Solution

I engineered an automated ETL pipeline that ingested raw data into a SQL database. I then developed a predictive Scikit-Learn model to forecast performance peaks and burnout risks.

The Outcome

Deployed an executive Power BI dashboard that visualized the ML predictions in real-time. This enabled data-backed, personalized training adjustments, reducing injury rates by 18%.

Technologies Used

Python | Scikit-Learn | SQL | Power BI | Data Engineering
