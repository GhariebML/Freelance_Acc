Hotel Billing System

Comprehensive database and analytics architecture for hospitality.

The Challenge

The hotel struggled with inefficient manual booking processes, leading to double-bookings, billing errors, and a complete lack of visibility into revenue trends and occupancy rates.

The Solution

I designed a robust, normalized relational database schema (Entity-Relationship diagram) and built a management analytics layer using complex SQL views to automate billing calculations.

The Outcome

Streamlined the reservation process entirely. Connected the SQL architecture directly to Power BI, providing management with real-time financial reporting and occupancy dashboards.

Technologies Used

PostgreSQL | Data Architecture | ETL | Power BI
